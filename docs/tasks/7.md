## Scope

- Use `rtrrl.py` as the standard experiment entrypoint.
- Create exactly four issue-scoped configs derived from `config/brax.yml`:
  - `brax-hopper` with `env_params.batch_size=64`
  - `brax-hopper` with `env_params.batch_size=1`
  - `brax-halfcheetah` with `env_params.batch_size=64`
  - `brax-halfcheetah` with `env_params.batch_size=1`
- Use Aim to save training results.
- Store RTRRL Aim runs in `logs/aim/.aim`.
- For Hopper runs, set `episodes: 10000` with `steps: 1000` so Aim reward
  curves reach `10m` on the logged step axis, and set `patience: 10000` so
  early stopping does not end the run before the target x-axis budget.
- Run one full pass in the authorized order:
  - Hopper `paral=64` on GPU
  - Hopper `paral=1` on CPU
  - HalfCheetah `paral=64` on GPU
  - HalfCheetah `paral=1` on CPU
